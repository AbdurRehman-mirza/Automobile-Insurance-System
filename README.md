# Automobile Insurance Management System

## Description
A console-based C++ application for managing automobile 
insurance operations including customers, vehicles, 
policies, claims and inspections.

## Features
- Customer & Vehicle Management
- Policy Selling
- Claim Filing & Approval
- Inspection Reports
- File-based data persistence (no DBMS)

## How to Run
1. Clone the repo
2. Open SDA_Project in Visual Studio
3. Default Manager login: MGR001 / admin123
4. Run and enjoy!

## Project Structure
include/    → Header files (.h)
src/        → Source files (.cpp)
data/       → Text files for data storage

## Design Principles
This project follows all 5 SOLID principles:
- S: Single Responsibility
- O: Open/Closed
- L: Liskov Substitution
- I: Interface Segregation
- D: Dependency Inversion

## Tech Stack
- Language : C++
- IDE      : Visual Studio
- Storage  : File-based (.txt)
